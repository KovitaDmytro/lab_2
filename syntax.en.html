JavaScript Syntax

JavaScript has a simple and flexible syntax, which includes:

  • Variable declaration
  • Function creation
  • Using operators
Operation Syntax Example
Variable Declaration let/const name; let x;
Assignment name = value; x = 10;

JavaScript Example:

script.js

// Display message in browser window
    alert('Hello, world!');
        
    // Change text of an element
    document.getElementById('myText').innerHTML = 'New text';
        
    // Add event to a button
    document.getElementById('myButton').addEventListener('click', function() {
        alert('Button clicked!');
    });
