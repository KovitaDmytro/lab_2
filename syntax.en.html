JavaScript Syntax

The JavaScript language has a simple and flexible syntax that includes:

  • Variable declarations
  • Function creation
  • Operator usage
Operation Syntax Example
Variable Declaration let/const name; let x;
Assignment name = value; x = 10;

Practical Examples

                    
// Example: Calculating factorial
function factorial(n) {
    if (n <= 1) return 1;
    return n * factorial(n - 1);
}
