Синтаксис JavaScript

Мова JavaScript має простий і гнучкий синтаксис, що включає:

  • Оголошення змінних
  • Створення функцій
  • Використання операторів
Операція Синтаксис Приклад
Оголошення змінної let/const ім'я; let x;
Присвоєння ім'я = значення; x = 10;

Приклад JavaScript:

script.js

// Вивід повідомлення у вікно браузера
    alert('Hello, world!');
        
    // Зміна тексту елемента
    document.getElementById('myText').innerHTML = 'New text';
        
    // Додавання події на кнопку
    document.getElementById('myButton').addEventListener('click', function() {
        alert('Button clicked!');
    });
